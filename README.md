# DatEdu - Online Course Platform

## Project Overview

DatEdu is an online course platform project developed for a Web Programming semester course. It is a PHP-based learning management system inspired by Udemy that allows users to browse, purchase, and enroll in online courses. The platform is built using plain PHP, HTML, CSS, and JavaScript with MySQL as the database backend, running on XAMPP with Apache.

## Technology Stack

- **Backend:** PHP (Plain PHP, no frameworks)
- **Frontend:** HTML5, CSS3, JavaScript
- **Database:** MySQL
- **Server:** Apache (via XAMPP)
- **Version Control:** Git

## Project Structure

The project is organized with the following main directories:

- `public/` - Public-facing pages and assets (CSS, JS, images)
- `app/` - Application logic (controllers, models, views, helpers)
- `config/` - Configuration files
- `database/` - Database schema and sample data
- `storage/` - Logs and temporary files
- `docs/` - Project documentation

## Getting Started

1. Extract the project to: `C:\BKU\Web_Programming\xampp\htdocs\Online_Course_Platform`
2. Start XAMPP and enable Apache and MySQL
3. Access the application at: `http://localhost/Online_Course_Platform/public/`
